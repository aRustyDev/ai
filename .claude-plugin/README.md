# Marketplace.json

> Marketplace catalog for claude plugins

- `https://ai.arusty.dev/`
  - `/plugins`: Marketplace of smart collections of agents, skills, commands, rules, and MCPs.
  - `/agents`: library of agents
  - `/skills`: library of skills
  - `/prompts`: library of prompts
  - `/commands`: library of commands
  - `/rules`: library of rules
  - `/output-styles`: library of output styles
  - `/mcp`: library of MCPs

- `https://mcp.arusty.dev/api/v1`
  - `/memory`: memory related MCP servers
    - `/plan`: plan related memories
    - `/`: plan related memories
  - `/search`: search related MCP servers
    - `/docs`: documentation related search endpoints
    - `/web`: documentation related search endpoints
